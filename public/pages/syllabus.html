Classes 1–3
Scratch Programming
  • Understand the Scratch workspace, sprites and costumes.
  • Basic sequencing: motion, looks and sound blocks.
  • Create simple animations and story-based projects.
  • Intro to events and simple loops via visual blocks.
Logical Thinking
  • Pattern recognition, sequences and problem decomposition.
  • Age-appropriate puzzles to build algorithmic thinking.
  • Visual activities to learn cause-effect (if → then).
Digital Literacy
  • Basic computer parts, safe device handling, and mouse/keyboard skills.
  • Intro to internet basics, searching and being safe online.
  • Typing fundamentals and simple productivity exercises (drawings, text).
Classes 4–6
Python Programming (Intro)
  • Foundations: variables, data types (strings, numbers), basic input/output.
  • Control flow: if/else, loops (for, while) with small exercises.
  • Simple functions and modular thinking.
  • Mini projects: calculator, number guessing game, quiz app.
Web Development (Basics)
  • HTML structure: headings, paragraphs, lists, images and links.
  • CSS basics: colors, fonts, simple layout and boxes.
  • Build & style a personal webpage or project showcase.
JavaScript & React (Intro)
  • Intro to JavaScript: variables, events and simple DOM changes.
  • Create interactive page elements: buttons, image sliders, simple games.
  • High-level idea of components (React concept) — no heavy tooling.
Data Science (Beginner)
  • What is data? Collecting, organizing and basic charting.
  • Use spreadsheets/Google Sheets to make charts and interpret results.
  • Mini project: analyze class survey or favorite books/games.
AI & Machine Learning (Awareness)
  • Everyday AI examples and interactive demos.
  • Simple classification tasks using visual tools or block-ML demos.
  • Creative projects exploring AI outputs (images/text).
Mobile Development (Intro)
  • Understand screens, buttons and app navigation concepts.
  • Build simple apps with visual tools (App Inventor/Thunkable).
  • Mini app: digital diary / counter / quiz app.
Robotics (Hands-on)
  • Introduction to simple robots and their parts.
  • Basic block-based programming for movements and sensors.
  • Project: simple line-following or obstacle-avoiding bot.
Cybersecurity (Safety)
  • Internet safety: personal info, privacy, and safe browsing.
  • Strong password practices and why they matter.
  • Simple, age-appropriate exercises on spotting scams.
Generative AI with ChatGPT (Intro)
  • What ChatGPT is and safe / creative uses for kids.
  • Use ChatGPT to write short stories, prompts, and learning aids.
  • Ethical basics: checking AI outputs and responsible use.
Classes 7–9
Python Programming (Intermediate)
  • Data structures: lists, tuples, dictionaries and sets with use-cases.
  • Functions, modules, file I/O and exception handling.
  • Object-Oriented basics: classes, objects and simple design patterns.
  • Working with libraries: requests, CSV, basic use of Pandas for tabular data.
  • Intermediate projects: student management system, basic automation scripts.
Web Development (Intermediate)
  • Responsive layouts (media queries), Flexbox and CSS Grid.
  • JavaScript fundamentals for DOM manipulation and events.
  • Build multi-page portfolio sites; introduce accessibility basics.
  • Deployment basics: host a static site using GitHub Pages or similar.
JavaScript & React (Intro → Applied)
  • ES6 features: let/const, arrow functions, template strings, destructuring.
  • DOM APIs, event handling and asynchronous basics (promises, fetch).
  • React fundamentals: components, props, state and simple routing idea.
  • Project: small SPA like todo list, weather widget consuming public API.
Data Science (Applied)
  • Introduction to Pandas: data frames, reading CSV, cleaning data.
  • Visualization with Matplotlib/Seaborn for insights.
  • Exploratory Data Analysis and simple hypothesis testing.
  • Mini projects: survey analysis, trend detection in datasets.
AI & Machine Learning (Foundations)
  • Supervised vs unsupervised learning concepts with examples.
  • Basic models: linear regression, decision trees, clustering.
  • Intro to model evaluation (accuracy, precision) and train/test split.
  • Hands-on mini ML project using scikit-learn (e.g., classify simple dataset).
Mobile Development (Practical)
  • UI/UX basics for mobile, responsive considerations, navigation patterns.
  • Build apps using Flutter or React Native (introductory projects).
  • Connect simple apps to local storage or mock APIs.
  • Project: notes app, basic chat UI or simple games.
Robotics (Intermediate)
  • Microcontrollers: Arduino / Raspberry Pi basics and setup.
  • Sensor integration (ultrasonic, IR) and actuator control (motors, servos).
  • Control logic, feedback loops, and simple automation tasks.
  • Project: autonomous robot with obstacle detection/line-following enhancements.
Cybersecurity (Intro to Practice)
  • Network basics, common threats (phishing, malware) and secure habits.
  • Intro to cryptography basics: hashes, symmetric/asymmetric idea.
  • Ethical hacking concepts and responsible disclosure mindset.
  • Practical labs: secure password checks, simple mock analysis.
Generative AI with ChatGPT (Applied)
  • Prompt engineering fundamentals and shaping model outputs.
  • Use ChatGPT to assist coding, research, and content generation.
  • Build small chatbot prototypes or content assistants (no heavy infra).
  • Ethics, hallucinations and verifying AI-generated information.
Classes 10–12
Python Programming (Advanced)
  • Advanced data structures and algorithms (big-O intuition, arrays, trees basics).
  • Advanced OOP: design, inheritance, polymorphism and design patterns.
  • Working with databases (SQL), REST APIs and building backend scripts.
  • Testing, debugging, packaging and deployment basics.
  • Capstone: end-to-end Python project (web-scraper + analytics or small web app).
Web Development (Full-stack Foundations)
  • Frontend advanced: React (hooks, context) and state management basics.
  • Backend essentials: Node.js/Express fundamentals, routing and middleware.
  • Databases: relational (Postgres/MySQL) and NoSQL concepts.
  • Authentication basics and deploying a simple full-stack app.
  • Project: portfolio + small full-stack application deployed online.
JavaScript & React (Advanced)
  • Advanced React patterns: hooks, context API, performance tips.
  • State management fundamentals (Redux concept or context + reducer).
  • Testing, linting and deployment workflows for frontend apps.
  • Project: build advanced SPA with API integration and auth flow.
Data Science (Advanced)
  • Data preprocessing pipelines, feature engineering and EDA best practices.
  • Machine learning models: regression, classification, cross-validation.
  • Model selection, evaluation metrics and basic hyperparameter tuning.
  • Visualization and dashboarding for insights; introduction to ML ops concepts.
  • Capstone: end-to-end data science project with report and presentation.
AI & Machine Learning (Advanced)
  • Deep learning fundamentals: neural networks, activation functions, training loops.
  • Practical training with TensorFlow or PyTorch (basic models & pipelines).
  • Model evaluation, overfitting, regularization and basics of deployment.
  • Project: image classifier or simple NLP pipeline.
Mobile Development (Advanced)
  • Advanced app architecture: state management, navigation patterns, API integration.
  • Native features: camera, GPS, notifications and offline storage.
  • Testing, debugging and publishing basics (store guidelines).
  • Project: fully functional mobile app with backend integration.
Robotics (Advanced)
  • Advanced sensors, computer vision basics (OpenCV) and motion planning.
  • ROS (Robot Operating System) introduction and simulation.
  • AI in robotics: reinforcement learning basics for robotics applications.
  • Project: advanced autonomous robot with perception and decision-making.
Cybersecurity (Advanced)
  • Network security, penetration testing fundamentals and vulnerability assessment.
  • Cryptography: practical encryption, hashing and digital signatures.
  • Incident response, digital forensics basics and security operations concepts.
  • Hands-on labs: CTF challenges, mock pen-test scenarios.
Generative AI with ChatGPT (Advanced)
  • Fine-tuning concepts and API integration for custom applications.
  • Building AI-powered applications: chatbots, content generators, code assistants.
  • Ethics, bias, and responsible AI deployment.
  • Capstone: custom application using generative AI APIs.
